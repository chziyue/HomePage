# My Home Page

个人导航主页项目，用于快速访问常用网站和服务。

## 功能特点

- 响应式设计，适配桌面和移动设备
- 网格布局，展示多个网站链接
- 美观的卡片式设计，包含网站图标和名称
- 悬停效果，提高用户交互体验
- 可自定义网站列表

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 使用方法

1. 要添加或修改网站链接，编辑 `index.html` 文件中的 `sites` 数组：
   ```javascript
   const sites = [
       { name: "网站名称", url: "网站URL", icon: "图标路径" },
       // 添加更多网站...
   ];
   ```

2. 将网站图标添加到 `img` 文件夹中。

## 自定义

- 修改 `<style>` 标签中的 CSS 来自定义页面样式。
- 调整媒体查询断点以适应不同的设备尺寸。
